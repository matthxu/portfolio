# Matthew Xu Portfolio

A single-page portfolio built with vanilla HTML, CSS, and JavaScript.

Live at [matthxu.com](https://matthxu.com).

## What This Includes

- Hero section with personal statement and quick links
- About section with background, education, and interests
- Work section with expandable project entries
- Skills section grouped by languages, frameworks, and tools
- Contact section with email and social links
- Custom cursor and reveal-on-scroll interactions

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Project Structure

- `index.html`: page structure and content
- `styles.css`: layout, typography, and responsive styling
- `scripts.js`: cursor, reveal animation, and project accordion behavior


## Run Locally

1. Clone or download the repository.
2. Open `index.html` in your browser.

Optional (recommended): use a local server in VS Code (for example, Live Server) for smoother development and refresh behavior.
